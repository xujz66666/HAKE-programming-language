\# Contributing Guide

\## Development Environment Setup



* Development Language: C (for speed)



* Compiler Version: GCC 9.0+



* Requirements: Easy to learn, fast execution. First, establish a solid foundation before deciding on development direction



* Editors: VS2022, CLion, or VSCode



* Operating System: Windows (start with solid basics, then expand to Linux/macOS)



\## Code Standards



Any style, writing comments is optional



\## Workflow



Claim tasks in Issues



Create feature branches from main branch



Complete changes and add tests



Submit Pull Request



\## Review Process



PR requires review by at least 1 core maintainer



All CI checks must pass



Modifications may be required, please cooperate patiently



\## Communication Guidelines



Report bugs in group chat



Propose new features in group chat



No profanity
